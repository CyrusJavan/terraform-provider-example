# Resource `example_pet`

Pet represents a single pet resource.



## Schema

### Required

- **name** (String, Required) Pet name.

### Optional

- **id** (String, Optional) The ID of this resource.
- **nicknames** (Set of String, Optional) Pet nicknames.
